== Description ==

Lets you insert a Google+ Page Badge to your site via shortcode. Easy to intall and implement.

Simply install the plugin and follow the instructions on the Settings page.

* [Google+ Page Badge](http://pleer.co.uk/wordpress/plugins/google-plus-page-badge/) Plugin homepage.
* More [WordPress Plugins](http://pleer.co.uk/wordpress/plugins/).

Simply use the shortcode [googlepb]


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `google-plus-page-badge` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Google+ Page Badge settings
4. Go through the steps and hit update
5. Insert the shortcode [googlepb] anywhere you want the badge to appear
